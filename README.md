# BlankOS Ultimate
Ferramenta avançada de otimização de sistema Windows desenvolvida por **carlos-blank**.

## Funcionalidades
- Limpeza profunda e reparo de sistema.
- Otimização de armazenamento (USB/HD/SSD) com blindagem anti-crash.
- Extrator de senhas de Wi-Fi.
- Modo de Desempenho Extremo.
- Sistema de Telemetria e Logs para diagnóstico de falhas.

## Como usar
1. Execute o script como **Administrador**.
2. Escolha uma das opções do menu.
3. Consulte o log gerado na sua Área de Trabalho em caso de erro.
