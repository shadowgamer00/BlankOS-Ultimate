Bem-vindo ao repositório oficial do **BlankOS**, uma ferramenta de otimização de sistema desenvolvida por **carlos-blank** para elevar o desempenho do seu Windows.

---

## 📋 Sobre o Projeto
O **BlankOS** é uma ferramenta de manutenção de nível avançado que automatiza tarefas críticas de limpeza e otimização. Ideal para utilizadores que procuram **gaming extremo**, **estabilidade de sistema** ou simplesmente **libertar espaço em disco** sem riscos.

## 🛠️ Menu de Funcionalidades
| Opção | Ação | Descrição |
| :--- | :--- | :--- |
| **1** | 🧹 Limpeza Profunda | Elimina lixo acumulado e cache de atualização. |
| **2** | 🛠️ Reparo do Windows | Corrige ficheiros de sistema (SFC/DISM). |
| **3** | 💿 Otimizar Discos | Otimiza a leitura e escrita (Defrag/TRIM). |
| **4** | 💾 Armazenamento | Repara Pens e HDs contra vírus de atalho. |
| **5** | 🌐 Rede e DNS | Reseta a tabela de rotas e limpa o DNS. |
| **6** | 🔍 Diagnóstico | Saúde física do disco e bateria. |
| **7** | 📡 Wi-Fi | Extrator de senhas salvas. |
| **8** | 🛡️ Sobrevivência | Finaliza processos travados e telemetria. |
| **9** | ⚡ Desempenho Extremo | Plano de energia de alta performance. |
| **10** | 🥔 Modo Batata | Otimização visual para FPS elevado. |

---

## 🚀 Como Utilizar
1. Faça o download do ficheiro **.bat**.
2. Clique com o botão direito e selecione **"Executar como administrador"**.
3. Siga o menu intuitivo.
4. **Nota:** O sistema gera automaticamente um relatório de log na sua Área de Trabalho caso ocorra algum erro.
